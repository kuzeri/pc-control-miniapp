# Abandoned Object Detector

Детектор оставленных предметов на видео — учебный проект по компьютерному зрению.

## Как работает

1. Загружаешь видеофайл (запись с камеры наблюдения, любое видео с людьми)
2. YOLOv8 детектирует **людей** и **предметы** (сумки, рюкзаки, чемоданы) на каждом кадре
3. Для каждого предмета проверяется, есть ли рядом человек (в радиусе N пикселей)
4. Если хозяин отошёл — запускается таймер
5. Когда таймер превышает порог — предмет помечается как **оставленный** (красная рамка + запись в журнал)

## Цветовая схема

- Зелёный — предмет рядом с хозяином
- Оранжевый — хозяин отошёл, идёт отсчёт  
- Красный — ТРЕВОГА, предмет без присмотра дольше порога

## Стек

- [YOLOv8](https://github.com/ultralytics/ultralytics) — детекция объектов
- [ByteTrack](https://github.com/ifzhang/ByteTrack) — трекинг через `model.track()`
- [Gradio](https://gradio.app) — веб-интерфейс
- OpenCV — обработка видео
